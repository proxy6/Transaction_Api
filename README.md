## Transaction API
- Technologies
  - Node js
  - Express
  - MySql
  - ORM (Knex)
  - Faker (Database Seed)

- How To Install and Run
 - Install Docker
 - Clone the Project
 - In your terminal run the command 
    - docker compose up -d
 - Note: Database is seeded with transaction data when docker is started
